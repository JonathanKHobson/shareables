Shareables MCP: The Shipping Lane

A non-sensitive preview of the local pipeline that turns approved project material into public HTML shareables.

What It Is

The Shareables MCP is the shipping lane for small public HTML artifacts. It keeps the manifest, local archive, style guide, validation checks, and GitHub Pages staging path in one place.

Codex or Claude can still write the page. The MCP makes the work repeatable: draft the manifest, build with a lens, save the HTML, validate it, stage it, then hand off the GitHub push.

1. Start

Create a manifest with the title, audience, purpose, sources, and listed or unlisted route.

2. Build

Use the recommended lens and locked Kyle style to turn approved material into one standalone HTML page.

3. Stage

Validate the page, add noindex for unlisted pages, copy it into the Pages repo, and get the publish handoff.

Safety Note

Unlisted does not mean private. It only means the page is excluded from the home page, public JSON index, and sitemap. Anything staged to GitHub Pages must be safe for public access by URL or repository browsing.
